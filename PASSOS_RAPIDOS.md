# 🚀 Passos Rápidos para Compartilhar o Repositório

## ✅ Verificação de Segurança
- ✅ `.gitignore` configurado corretamente
- ✅ `serviceAccountKey.json` está protegido
- ✅ `.env` está protegido

## 📝 Próximos Passos

### 1. Criar Repositório no GitHub
1. Acesse: https://github.com/new
2. Nome: `TeamHIIT-APP`
3. Marque como **Private**
4. Clique em **"Create repository"**

### 2. Conectar e Enviar Código
Execute no terminal (na pasta do projeto):

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/TeamHIIT-APP.git
git push -u origin main
```

**Substitua `SEU_USUARIO` pelo seu username do GitHub**

### 3. Adicionar Colaboradores
1. No repositório: **Settings** → **Collaborators**
2. **Add people**
3. Digite o email/username do freelancer
4. Permissão: **Read** (apenas leitura para avaliação)
5. Repita para o segundo freelancer

### 4. Enviar Convites
Os freelancers receberão um email de convite automaticamente.
