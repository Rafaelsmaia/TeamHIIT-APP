## 🎯 O que você precisa fazer agora:

### 1. Criar Repositório no GitHub (2 minutos)

1. Acesse: **https://github.com/new**
2. Nome: `TeamHIIT-APP`
3. Marque como **Private**
4. Clique em **"Create repository"**

### 2. Executar Scripts (1 minuto)

Abra o terminal na pasta do projeto e execute:

```bash
# Opção 1: Script automático (mais fácil)
node scripts/push-to-github.js SEU_USUARIO_DO_GITHUB

# Exemplo:
node scripts/push-to-github.js rafael-maia
```

**OU** execute manualmente:

```bash
git init
git add .
git commit -m "Initial commit - Team HIIT App"
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/TeamHIIT-APP.git
git push -u origin main
```

### 3. Adicionar Colaboradores (2 minutos)

1. No repositório: **Settings** → **Collaborators**
2. **Add people** → Digite email/username do freelancer
3. Permissão: **Read**
4. Repita para o segundo freelancer

### 4. Enviar Links

Envie para cada freelancer:
- Link: `https://github.com/SEU_USUARIO/TeamHIIT-APP`
- Instrução: "Aceite o convite enviado por email"
